# Platform Roadmap

This roadmap outlines the architectural phases of the Orchestra platform.

## 🟢 Phase 1: Core Foundation (Complete)
- **Secure Electron/React Bridge**: Native-to-Web communication layer.
- **High-Density Dashboard**: Command center for fleet operations.
- **Real-time SSE Sync**: Sub-100ms state updates.

## 🟡 Phase 2: Operator Experience (Complete)
- **Visual Kanban Board**: Drag-and-drop issue management.
- **Issue Inspector**: Live log streaming and diff browser.
- **Unified Knowledge Base**: Integrated wiki and technical specs.

## 🔵 Phase 3: Agent Intelligence (Complete)
- **Global Control Plane**: Direct management of agent dotfiles.
- **Skill Injection**: Behavioral templates for strategic guidance.
- **Model Context Protocol (MCP)**: Dynamic tool host integration via JSON-RPC.
- **Multi-Agent Cascading**: Automated provider switching after repeated failures.

## 🔴 Phase 4: Enterprise & Scale (In Progress)
- **Interactive Dmux Terminals**: Multi-pane real-time shell orchestration. (Complete)
- **Human-In-The-Loop (HITL)**: Direct PTY intervention for automated sessions. (Complete)
- **Extensible Tools**: Dynamic MCP tool schema discovery and resource support. (Complete)
- **Stability Analytics**: Project health scoring and trend forecasting. (Planned)
- **Automated Maintenance**: Background workspace garbage collection and artifact cleanup. (Planned)
